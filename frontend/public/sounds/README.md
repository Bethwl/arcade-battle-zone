# Arcade Sound Effects

This directory contains retro arcade sound effects for the FHE Rock-Paper-Scissors game.

## Required Sound Files

You need to add the following 8 sound files to this directory:

1. **click.mp3** - Button click sound
2. **select.mp3** - Move selection sound
3. **submit.mp3** - Move submission sound
4. **game-created.mp3** - Game created success sound
5. **player-join.mp3** - Player joins sound
6. **game-start.mp3** - Game start sound
7. **winner.mp3** - Winner reveal sound
8. **error.mp3** - Error sound

## Free Sound Resources

### Recommended Sources:
- **Freesound.org** - Search: "8-bit button", "arcade beep", "retro game"
- **OpenGameArt.org** - Arcade sound packs
- **BFXR.net** - Generate custom retro sounds online (free tool)
- **Zapsplat.com** - Free game sounds (attribution required)
- **Kenney.nl** - Free game assets including sounds

### Sound Specifications:
- **Format**: MP3 or OGG
- **Length**: 50-200ms for UI sounds, 1-2s for jingles
- **Sample rate**: 22kHz or 44kHz
- **Bit rate**: 128kbps

## Quick Start with BFXR:
1. Go to https://www.bfxr.net/
2. Click different preset categories (Pickup/Coin, Laser/Shoot, etc.)
3. Randomize until you find sounds you like
4. Export as WAV, then convert to MP3
5. Rename to match the filenames above
